# Event Management Service
## Description

A a backend service for event management that facilitates the creation, management, and eventual automatic deletion of events.

## Installation

```bash
$ pnpm install
```

## Running the app

```bash
# development
$ pnpm run start

# watch mode
$ pnpm run start:dev

# production mode
$ pnpm run start:prod
```

## Test

```bash
# unit tests
$ pnpm run test

# e2e tests
$ pnpm run test:e2e

# test coverage
$ pnpm run test:cov
```

## Linting and formatting

```bash
# Lint and autofix with eslint
$ pnpm run lint

# Format with prettier
$ pnpm run format

```
## Start Docker

```bash
$ docker-compose up

```
